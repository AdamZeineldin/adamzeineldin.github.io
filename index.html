BigBrother

Nov 2025 – Present

An AI-powered memory assistant web app that detects motion, records synchronized video and audio, and uses Google Gemini to summarize and search your daily events — all running locally with no cloud reliance.

Key Features

  • Real-time motion detection and recording pipeline using OpenCV and YOLOv8 to log accurate event clips
  • Google Gemini 2.5 Flash integration for video summaries, speech-to-text, image captions, and natural language search
  • Python + Flask back-end with a React front-end for a clean, responsive UI
  • Fully local SQLite database and memory-node system — no cloud, complete privacy

Jun 2025 – Aug 2025

A Q-learning agent that learned optimal Tic Tac Toe strategies entirely through self-play, with no hard-coded rules — reaching consistent optimal play after 100,000 training iterations.

Key Features

  • Implemented a Q-learning agent with Q-values and reward systems to guide both players toward optimal strategies
  • Tuned hyperparameters (learning rate, discount factor, epsilon) to balance exploration and exploitation
  • Trained across 100,000 self-play iterations with no human-authored game logic
  • Reached consistent optimal play against both rule-based and perfect-play opponents
