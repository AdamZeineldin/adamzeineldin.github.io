TerpGigs

2024 – Present

A full-stack peer-to-peer gig marketplace built exclusively for University of Maryland students. Students can post requests for help with tasks like moving, tutoring, tech support, and more — and other Terps can accept and earn money.

Key Features

  • Real-time chat between request poster and acceptor via Firebase Firestore
  • Firebase Authentication with user profiles
  • Request lifecycle management (open → accepted → completed)
  • Responsive design with Tailwind CSS
  • Deployed on Vercel with Next.js App Router
